# 🎮 Game Design Document — STARWARS: AR Space Fighter

> Версия документа: 1.0 · Статус: активный · Платформа: WebXR (мобильный браузер)

---

## 1. Высокая концепция (High Concept)

**STARWARS** — это AR-шутер от первого лица, в котором игрок становится пилотом
космического истребителя в центре масштабного сражения с инопланетным флотом.
Подняв телефон, игрок видит поверх реального окружения звёздное поле боя:
вражеские корабли налетают со всех сторон, и нужно поворачиваться всем телом,
наводить прицел гироскопом и вести огонь, переживая нарастающие волны врагов.

**Жанр:** Аркадный космический шутер (rail-free, 360° AR).
**Сеттинг:** Космическое сражение, кинематографичный sci-fi реализм.
**Аудитория:** Казуальные и кор-игроки, любители AR-новинок, 12+.
**Сессия:** 3–7 минут на забег (идеально для мобайла).
**Платформа:** Мобильный браузер (Android Chrome, iOS Safari), без установки.

### Pillar-ы дизайна (3 столпа)

1. **«Ты внутри битвы».** AR + 360°-наведение создают физическое ощущение
   присутствия. Игрок крутится в реальном пространстве — враги действительно
   повсюду.
2. **«Кинематографичность каждую секунду».** Bloom, трассеры, объёмные взрывы,
   адаптивная музыка — каждое попадание ощущается эпично.
3. **«Просто взять, сложно отложить».** Один тап = выстрел. Мгновенный вход,
   комбо-петля и рекорды держат «ещё один забег».

## 2. Игровой цикл (Core Gameplay Loop)

```
   ┌─────────────────────────────────────────────────────┐
   │  ОСМОТРИСЬ (гироскоп/свайп) → НАВЕДИ прицел на врага  │
   │            ↓                                           │
   │  ОГОНЬ (тап) → ПОПАДАНИЕ → взрыв + очки + комбо       │
   │            ↓                                           │
   │  УКЛОНИСЬ от вражеского огня (поворот/буст)           │
   │            ↓                                           │
   │  ВОЛНА ОЧИЩЕНА → награда → следующая волна (сложнее)  │
   │            ↓                                           │
   │  БОСС каждые N волн → ПОБЕДА / ПОРАЖЕНИЕ → рекорд      │
   └─────────────────────────────────────────────────────┘
```

**Микро-цикл (секунды):** прицелься → стреляй → попади → уклонись.
**Макро-цикл (минуты):** очисти волну → стань сильнее/быстрее → одолей босса.
**Мета-цикл (между забегами):** побей рекорд → открой следующую сложность.

## 3. Управление

| Действие | Ввод | Примечание |
|---|---|---|
| Поворот обзора / наведение | DeviceOrientation (гироскоп) | Основной способ в AR |
| Поворот обзора (фолбэк) | Свайп пальцем по экрану | Если гироскоп недоступен/запрещён |
| Огонь | Тап по экрану / кнопка FIRE | Поддержка авто-огня при удержании |
| Буст / рывок-уклонение | Кнопка BOOST | Кулдаун, кратко гасит входящий урон |
| Пауза | Кнопка ⏸ | Останавливает симуляцию и аудио |

Прицел **зафиксирован в центре экрана**. Игрок наводит, поворачивая устройство.
Это снимает проблему «двух стиков» на мобиле и усиливает AR-погружение.

## 4. Механики

### 4.1 Стрельба
- **Бластеры:** мгновенные снаряды-трассеры из двух пушек истребителя.
- **Скорострельность:** базовая, повышается перегревом? Нет — простой кулдаун + пул.
- **Прицеливание-помощник (aim assist):** лёгкий магнетизм к ближайшему врагу в
  конусе прицела (настраивается, см. `config.js`).

### 4.2 Здоровье / щиты
- У игрока **щит** (регенерирует вне боя) и **корпус** (не регенерирует за забег).
- Попадание врага → урон по щиту; при пустом щите — по корпусу.
- Корпус = 0 → поражение.

### 4.3 Комбо и счёт
- Каждое убийство даёт очки × **комбо-множитель**.
- Комбо растёт за серию убийств без получения урона; сбрасывается при попадании по
  игроку или по таймауту.
- Рекорд хранится в `localStorage`.

### 4.4 Буст / уклонение
- Краткий рывок: повышенная манёвренность камеры + временная неуязвимость (i-frames).
- Кулдаун, чтобы не спамить.

### 4.5 Волны и прогрессия
- Волна = набор врагов; очистил всех → следующая волна.
- С каждой волной растут: число врагов, их скорость, частота и точность огня.
- **Босс-волна** каждые `N` волн (см. config): крупный корабль с фазами и слабыми
  точками.

## 5. Враги

| Тип | Поведение | Угроза | Очки |
|---|---|---|---|
| **Scout (Разведчик)** | Быстрый, прямые наскоки, слабый | Низкая | 100 |
| **Interceptor (Перехватчик)** | Кружит, стреляет очередями, уклоняется | Средняя | 250 |
| **Bomber (Бомбардировщик)** | Медленный, толстый, тяжёлый залп | Высокая | 400 |
| **Carrier / Boss (Носитель)** | Стационарно-маневрирующий, спавнит мелочь, фазы | Босс | 5000 |

**Боевой ИИ (см. `systems/EnemyAI.js`):** состояния `approach → strafe → fire →
reposition`. Перехватчики уклоняются от трассеров игрока, бомбардировщики прут напролом.

---

## 6. Прогрессия и баланс

- **Кривая сложности:** линейный рост врагов на волну + мягкий рост скорости/урона.
  Параметры в `config.js` (`difficulty`).
- **Награды за волну:** восстановление части щита, рост комбо-потолка.
- **Длина забега:** ~8–12 волн до первого носителя; забег ~5–7 минут.

---

## 7. UX / экраны

1. **Брифинг (Start):** лор + инструкция управления + большая кнопка «ENGAGE».
   Здесь же запрашиваются разрешения камеры и сенсоров (требование iOS — по жесту).
2. **Игра:** AR-фон + космос + HUD (прицел, щит, корпус, счёт, комбо, волна, мини-радар).
3. **Пауза:** затемнение, продолжить/выйти.
4. **Поражение (Game Over):** счёт, рекорд, «Retry».
5. **Победа волны / Победа над боссом:** короткий триумф-экран, переход дальше.

### Радар (мини-карта)
Круговой индикатор показывает направление на ближайших врагов вне поля зрения —
критично для 360°-боя, чтобы игрок знал, куда повернуться.

## 8. Сеттинг и лор (кратко)

> Год 2387. Колониальный флот Земли разбит у пояса Хелиос. Последняя линия обороны —
> одиночные пилоты «Вэнгард». Рой Сайленов — кремниевая инопланетная цивилизация —
> вышел из варпа прямо над домашней станцией. Ты — крайний рубеж. Удержи небо.

Лор подаётся короткими строками на брифинге и в подсказках между волнами. Не мешает
аркадному темпу.

---

## 9. Аудио (резюме; полно — в AUDIO_DESIGN.md)

- Процедурный синтез: лазеры, взрывы, гул двигателя, попадания, UI-клики.
- Адаптивная музыка: спокойный слой на брифинге → боевые слои нарастают с волнами.
- Пространственный звук (PannerNode) — взрывы и враги звучат с их направления.

---

## 10. Метрики успеха (для вертикального среза)

- Стабильные **60 FPS** на среднем мобиле (или 30 как минимум).
- Время до первого выстрела от загрузки **< 15 c**.
- «Читаемость» боя: игрок всегда понимает, откуда угроза (радар + 3D-звук).
- Забеги завершаются (нет софт-локов), рекорд сохраняется между сессиями.
