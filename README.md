## 🎬 Что это

**STARWARS** — это браузерная **AR-игра** в жанре космического шутера от первого лица.
Поднимаешь телефон — и вокруг тебя, поверх живого изображения с камеры, разворачивается
звёздное поле боя: вражеский флот пришельцев, лазерные залпы, объёмные взрывы и
кинематографичная музыка. Ты — пилот. Поворачивайся всем телом, целься гироскопом,
жми гашетку.

Никаких установок: игра работает прямо в мобильном браузере (Android Chrome / iOS Safari).

## ✨ Ключевые особенности

- 🪐 **AR-lite трекинг** — фид с задней камеры как фон + управление наведением через
  гироскоп устройства. Космос накладывается на реальное пространство.
- 🚀 **Кинематографичная графика** — процедурное звёздное небо, объёмная туманность на
  шейдерах, HDR-bloom постобработка, частицы взрывов, неоновые трассеры лазеров.
- 👾 **Волны врагов с боевым ИИ** — несколько типов истребителей пришельцев, манёвры
  уклонения, боссы каждые несколько волн.
- 🔊 **Процедурный 3D-звук** — лазеры, взрывы, гул двигателей и адаптивная боевая музыка
  синтезируются в реальном времени через Web Audio API (без тяжёлых аудиофайлов).
- 🎯 **AAA-HUD** — прицел, щиты, счёт, комбо, индикатор волны, экраны брифинга/победы/
  поражения.
- 📈 **Прогрессия** — нарастающая сложность, комбо-множитель, рекорд в localStorage.

## 🎮 Как играть

| Действие | Управление |
|---|---|
| Осмотреться / прицелиться | Поворачивай **устройство** (гироскоп) или **свайп** по экрану |
| Огонь | **Тап** по экрану или кнопка **FIRE** |
| Буст / уклонение | Кнопка **BOOST** на HUD |
| Пауза | Кнопка ⏸ в углу |

> Уничтожай пришельцев, держи комбо, не дай щитам упасть до нуля. Переживи все волны и
> победи флагман-носитель.

## 🚀 Запуск локально

Игре **не нужен билд** — это статические ES-модули, Three.js грузится с CDN.

```bash
# Любой статический сервер. Например:
npx serve . -l 8080
#   ИЛИ
python3 -m http.server 8080
```

Открой `http://localhost:8080`.

> ⚠️ **AR (камера + гироскоп) требует HTTPS** (secure context). На `localhost` камера
> работает, но для теста с телефона по сети нужен HTTPS-туннель (`ngrok`, `cloudflared`)
> или деплой на Netlify (см. ниже).

## ☁️ Деплой

Проект готов к деплою на **Netlify** как есть (`netlify.toml` уже настроен — статика,
HTTPS, нужные `Permissions-Policy` для камеры и сенсоров). Просто подключи репозиторий
или перетащи папку в Netlify Drop.

## 🧱 Технологический стек

- **Three.js** (r160) — WebGL рендеринг, постобработка (EffectComposer + UnrealBloom).
- **WebXR / DeviceOrientation API** — AR-наведение по гироскопу.
- **getUserMedia** — фид с камеры как AR-фон.
- **Web Audio API** — процедурный синтез звука и музыки.
- **Vanilla ES Modules** — без бандлера, без зависимостей в рантайме (кроме CDN Three.js).
